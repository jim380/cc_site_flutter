## Develop
```bash
$ flutter channel beta
$ flutter upgrade
$ flutter config --enable-web 
$ flutter run -d chrome
```

## Build
```bash
$ flutter build web
```

## Host
Copy over the `build/web` folder and NGINX it. You can also use other hosting services such as Github Pages and Firebase Hosting.
